Codex Browser Relay

Click the toolbar button on a tab to attach or detach the CDP bridge.

What this extension does

This extension connects normal web pages to a local relay so external tools can inspect text, click elements, type into inputs, and coordinate page automation.

For regular sites it uses a page bridge approach first, which is more robust in Edge. Internal browser pages still cannot be controlled by extension policy.

Relay port

Default: 18793. Extension connects to: http://127.0.0.1:<port>/.

Protocol expected from the backend

HTTP health check on HEAD /.

WebSocket endpoint on /extension.

JSON messages using ping, pong, pageCommand, pageAttached and pageDetached.
